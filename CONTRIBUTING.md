# Contributing

1. Fork or branch from `main` / `master` (per your org’s workflow).
2. Create a virtualenv: `python3 -m venv .venv && source .venv/bin/activate`.
3. Install: `pip install -r requirements.txt`.
4. Run tests: `make test` or `pytest tests/ -v`.
5. Open a pull request with a short description of changes.

This project is for **authorized security lab** use. Do not add exploit payloads or unauthorized scanning logic.
