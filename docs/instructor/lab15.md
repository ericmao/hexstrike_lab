## 預期產物（依 15A–15D 擇一）

| 選項 | 核心驗收 |
|------|-----------|
| **15A** | 雲端／容器工具摘要 + RoE 合規（無越權帳號） |
| **15B** | 公開 BBY 計畫之檢核清單；**無**未授權流量 |
| **15C** | 偵測規則／告警邏輯 + FP 討論 |
| **15D** | MCP 與 `hexstrike_lab pipeline` 橋接設計或原型；可稽核、不越權 |

## 評分對照

| 等級 | 要點 |
|------|------|
| **及格** | 與所選子題一致；有範圍與 RoE 自省。 |
| **優良** | 15D：能說明錯誤隔離與 secret／target 不進 log 的作法。 |

## 常見不合格徵兆

- 15B 對真實程式送測；15A 使用真實客戶環境無授權證明。
