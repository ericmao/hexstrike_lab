# Lab 12 — RoE 治理 — 講師專用大綱

**學員教材：** [`labs/lab12/cursor_prompts.md`](../../labs/lab12/cursor_prompts.md)  
**回索引：** [`INSTRUCTOR_ANSWER_OUTLINE.md`](../INSTRUCTOR_ANSWER_OUTLINE.md)

## 預期產物

- 完整 RoE：範圍（CIDR／URL）、時段、允許／禁止技術、聯絡窗口、中止條件、證據保存。
- 互評：**3 個**模糊地帶之澄清問句。
- 一句話：本課程最危險的誤用情境。

## 評分對照

| 等級 | 要點 |
|------|------|
| **及格** | 白名單與禁止項皆非空；無「整家公司網路」式模糊句而無列舉。 |
| **優良** | 明寫 **human-in-the-loop**：何種 MCP 操作必須人工核准。 |

## 常見不合格徵兆

- 禁止項留白；無授權人／期間。
- 互評問句敷衍（未觸及密碼噴射、DoS、資料外洩等）。

## 口試參考

- AI 或代理能否「代替」授權人簽署 RoE？法律與程序上答案為何？
