## 預期產物

- Markdown 滲透報告（章節齊備）。
- `hexstrike_lab pipeline` 產物路徑證明（`report.json` 或 `manifest.json`）。
- 半頁對照：機械 findings vs 質性敘事。
- Track C：三格 attack-chain 表；選修 MITRE ID／flag。

## 評分對照

| 等級 | 要點 |
|------|------|
| **及格** | **每一報告章節**能指到 log／檔案／輸出；目標僅講師發放 IP。 |
| **優良** | 能指出代理幻覺（宣稱執行但無 log）並修正敘述。 |

## 常見不合格徵兆

- 泛用模板、無標的專屬證據。
- Phase 與實際輸出時間線不一致。
